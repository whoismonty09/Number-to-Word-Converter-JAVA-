# 🔢 Number to Word Converter (Java)

A simple Java GUI application that converts numeric values into their word representation using Java Swing.

---

## 📌 Description

The Number to Word Converter is a Java-based GUI application that allows users to enter a number and converts it into words. It demonstrates Java Swing GUI development and logical programming concepts.

---

## 🚀 Features

- 🔢 Converts numbers into words
- 🖥️ Simple and clean GUI
- ⚡ Instant conversion
- 🧠 Logical number processing
- 💡 Easy to use

---

## 🛠️ Technologies Used

- Java  
- Java Swing (GUI)  
- Event Handling  
- Object-Oriented Programming  

---

## ▶️ How to Run

1. Install Java JDK.
2. Open the project in VS Code or any Java IDE.
3. Compile the program:
   ```bash
   javac NumberWordConverter.java
